gulp skeleton
=============


Getting started
---------------

```
npm install -g gulp
npm install
```


Run tasks
---------

### Build

#### Production
CSS minified and JavaScript uglified.

```
gulp
```

or `gulp build`, `gulp build:production`


#### Development

```
gulp build:development
```


### Watch

```
gulp watch
```

alias is `gulp w`


### And more

```
gulp --tasks-simple
```
